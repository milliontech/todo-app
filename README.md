# todo-app
A simple android app created by Apache Cordova and Vue.js

## Step 1: Prepare the project

### Read cordova-sample\README.md
```
cd cordova-sample
npm ci
npx cordova prepare
```

### Read vue-sample\README.md
```
cd ..\vue-sample
npm ci
```

## Step 2: Build the app
```
npm run build
```

## Step 3: Run the app
```
npm run android
```
